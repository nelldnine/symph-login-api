### Installation

1. Install Flask to `lib` folder by entering the following command to the terminal `pip install -t lib Flask`.
2. After installation, run `dev_appserver.py` to the terminal.
3. Enjoy :)

### Login

Endpoint
```
/login
```

Body
```
email - String
password - String
```

### Register

Endpoint
```
/register
```

Body
```
email - String
password - String
name - String
```
